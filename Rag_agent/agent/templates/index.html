Welcome to the RAG Based Chatbot Project

Providing users with a platform to get answers to questions through the context provided by any kind of website URL or YouTube video.

This project aims to solve the problem of LLMs where they can't keep up with the latest information. Using Retrieval Augmented Generation, we solve the Needle in the Haystack problem, ensuring accurate information retrieval and reducing misinformation.
